### Story 04-01: 智能仓位管理算法
**优先级**: P0
**故事点**: 5
**状态**: 待开发

**用户故事**:
作为风险管理员，我需要智能仓位管理系统，以便根据信号信心度和市场趋势动态调整仓位大小。

**验收标准**:
- [ ] 实现基础仓位计算 (100 USDT 基数)
- [ ] 实现信心度分级仓位:
  - HIGH: 1.5x
  - MEDIUM: 1.0x
  - LOW: 0.5x
- [ ] 实现趋势强度调整 (1.2x)
- [ ] 实现最大仓位限制 (10%)
- [ ] 实现动态调整算法
- [ ] 支持参数配置
- [ ] 单元测试验证

**任务分解**:
- [ ] 设计仓位计算算法
- [ ] 实现分级调整逻辑
- [ ] 实现趋势强度因子
- [ ] 添加风险控制检查
- [ ] 编写测试用例

**技术细节**:
- 文件路径: `src/execution/position_manager.py`
- 计算公式:
  ```
  position_size = base_amount × confidence_multiplier × trend_multiplier
  position_size = min(position_size, max_usdt × max_position_ratio)
  ```
- 基础: 100 USDT
- 最大: 10% 账户资金
